# 🚇 Metro Navigation System

A web-based Metro Navigation System that helps users find routes, calculate distance, and visualize metro stations using an interactive map.

---

## 🌟 Features

- 🚇 Route finding using Dijkstra Algorithm
- 📏 Distance calculation between stations
- 🗺️ Interactive map using Leaflet.js
- 🎥 Video background UI (modern design)
- 🔊 Voice announcement system
- 📊 Dynamic station data using JSON

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Leaflet.js (Map Integration)
